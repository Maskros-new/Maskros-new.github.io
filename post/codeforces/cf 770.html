B_Fortune_Telling (xor)

题意：

给定一个 数组，可以对一个非负整数 遍历数组 (从 1 到 n) 做如下操作: 或者 。

Alice 从 开始，Bob 从 开始，已知输入数据一定合法，询问 Alice 和 Bob 谁能遍历数组后得到

思路：

重要性质： 和 的奇偶性不会改变

根据题意， 和 的奇偶性固然不同，遍历判断奇偶性即可。

void solve() {
    ll n, x, y; cin >> n >> x >> y;
    x = x & 1 ? 1 : 0;
    rep (i, 0, n){
        int t; cin >> t; 
        x = (x + t) & 1 ? 1 : 0;
    }
    if ((x & 1) == (y & 1)) puts("Alice");
    else puts("Bob");
}
